BIO-IK

Install:

cd NICO-software/api/src/nicomoveit
git clone https://github.com/TAMS/bio_ik.git
cd NICO-software/api/
catkin_make

Usage:

cd NICO-software/api/src/nicomoveit
python setKinematicsSolver.py bio_ik

OR

When using the moveitWrapper the IK solver can be changed using the function

OR

gedit NICO-software/api/src/nicomoveit/moveitgenerated/config/kinematics.yaml
Replace kinematics_solver: kdl_kinematics_plugin/KDLKinematicsPlugin (or similar) with kinematics_solver: bio_ik/BioIKKinematicsPlugin
