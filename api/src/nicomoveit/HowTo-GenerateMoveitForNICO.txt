If the directory /NICO-software/api/src/nicomoveit with previous MoveIt! setup does not exist:
---------------------------------------------------------------------------------------------------
1. Get NICO repository
2. cd ~/NICO-software/api
3. source NICO-setup.bash or if this doesn't work source /opt/ros/kinetic/setup.bash, catkin_make, source devel/setup.bash
4. cd ~/NICO-software/api/src
5. mkdir nicomoveit
6. cd nicomoveit

Not necessary anymore 7. catkin_create_pkg moveitmeshes
Not necessary anymore 8. cd moveitmeshes
Not necessary anymore 9. mkdir collisionmeshes
Not necessary anymore 10. Copy meshes into the created folder

11. cd ~/NICO-software/api/src/nicomoveit
12. catkin_create_pkg moveiturdf
13. cd moveiturdf
14. mkdir urdf
---------------------------------------------------------------------------------------------------
If the directory /NICO-software/api/src/nicomoveit with previous MoveIt! setup does already exist:

source NICO-setup.bash or if this doesn't work source /opt/ros/kinetic/setup.bash, catkin_make, source devel/setup.bash
---------------------------------------------------------------------------------------------------
15. execute script~/NICO-software/api/src/nicomoveit/createMoveitUrdf.py
16. roslaunch moveit_setup_assistant setup_assistant.launch
17. Create New MoveIt Configuration Package -> Browse -> ~/NICO-software/api/src/nicomoveit/moveiturdf/urdf/complete.urdf -> Open -> Load Files
18. Self-Collisions -> Sampling Density to high -> Regenerate Default Collision Matrix
19. Planning Groups -> Add Group
Group Name: leftArm
Kinematic Solver: kdl_kinematic_plugin/KDLKinematicsPlugin
-> Add Joints -> select all six joints from l_shoulder_z to l_wrist_x -> Press '>' -> Save
20. Same for rightArm, leftLeg (l_hip_z to l_ankle_x) and rightLeg
21. Author Information -> Set Name and Email
22. Configuration Files -> Browse -> ~/NICO-software/api/src/nicomoveit -> Create Folder -> moveitgenerated -> Press Enter -> Open -> Generate Package -> Ok -> Exit Setup Assistant
23. cd ~/NICO-software/api/src/nicomoveit/moveitgenerated/launch
24. add this line "<include file="$(find kinematics)/launch/kinematics_server.launch" />" to demo.launch
25. cp demo.launch nico.launch
26. remove this part in nico.launch
  <!-- Run Rviz and load the default config to see the state of the move_group node -->
  <include file="$(find moveitgenerated)/launch/moveit_rviz.launch">
    <arg name="config" value="true"/>
    <arg name="debug" value="$(arg debug)"/>
  </include>
27. cd ~/NICO-software/api/
28. source NICO-setup.bash 

To change something on the MoveIt! configuration you can call:
  roslaunch moveit_setup_assistant setup_assistant.launch
  ->  Edit Existing MoveIt Configuration Package
  -> Browse -> ~/NICO-software/api/src/nicomoveit/moveitgenerated/ -> Open -> Load Files
  
Because the MoveIt! package has been downloaded the Setup Assistant will say that files have been modified, to change these files they have to be marked manually
