Install:
- if you use ROS Kinetic (Ubuntu 16) 
  sudo apt-get install ros-kinetic-moveit
  if you use ROS Indigo (Ubuntu 14)
  sudo apt-get install ros-indigo-moveit
- pip install --user pyassimp --upgrade   (For pyassimp-3.3)

Usage:
First terminal
  source NICO-setup-Moveit.bash  
  roscore
  
OPTIONAL second terminal to use RViz visualization:
  roslaunch moveitgenerated demo.launch
 
Third terminal
  source NICO-setup-Moveit.bash  
  cd examples/nicomoveit
  python moveitKinematicsTest.py  
  python moveitPathPlanningTest.py
