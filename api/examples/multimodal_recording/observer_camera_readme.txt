How to start the observer camera server on wtmpc211:

- login as sysadmin

in first terminal 
- source /opt/ros/kinetic/setup.bash
- roslaunch realsense_camera r200_nodelet_default.launch

in second terminal

- change to directory /export/NICO/NICO-software/api/examples/multimodal_recording
- start ./start_observer_server.sh
